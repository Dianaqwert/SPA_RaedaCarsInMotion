# RAEDA Cars in Motion 🚗💨

**RAEDA Cars** is a full-stack web application designed for managing customer requests and services in a hypothetical automotive agency. The system functions both as an informative *landing page* and as an interactive platform for registered users.

🔗 **Live Site:** [https://spa123hs.web.app/](https://spa123hs.web.app/)  
💻 **Repositories:** [Frontend (Angular)](https://github.com/Dianaqwert/SPA_RaedaCarsInMotion) | [Backend (Node.js)](https://github.com/Dianaqwert/BackEndApi_nodeJS)

---

## Key Features

* **User Management:** Registration and login via email/password and authentication with **Google**.  
* **Interactive Catalog:** Vehicle browsing with a functional search by name, model, or brand.  
* **Services Module:** Reactive form for requesting maintenance, repairs, or diagnostics, with real-time validations.  
* **Financing Calculator:** Tool to calculate interest and monthly payments based on loan amount and term.  
* **Advanced Web Accessibility:** Specialized menu including text-to-speech (TTS), font size adjustment, and contrast switching (Dark Mode).  
* **QR Generation:** System for claiming rewards through dynamic QR codes.  
* **PWA (Progressive Web App):** Support for installation and optimized performance on mobile devices.

## Development & Setup

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 19.2.11.

### Development Server
To run the local server:
```bash
ng serve
```
Navigate to http://localhost:4200/. The app will automatically reload if you change source files.
### Build
To compile the project:
```bash
  ng build
```
The output files will be stored in the dist/ directory.

This system represents the final project for the Web Technologies course, taught by Dr. Georgina Salazar Partida. The objective was to apply skills in responsive design, asynchronous communication (HTTP), and cloud service integration.
